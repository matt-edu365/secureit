# Legacy Shared-Host Prototype Source

This folder contains the original PHP prototype source used for the shared-host deployment on `example.ict365.uk`.

## Important

This is no longer the primary application surface.

The current container-ready SecureIT app lives in:
- `app/`

The deployable shared-host bundle currently lives in:
- `deploy/maester/`

## Why this folder still exists

It is being kept temporarily so the live prototype can remain available while SecureIT is refactored for Docker, GHCR, and Proxmox deployment.

## Intended future state

Once the Docker-based SecureIT deployment replaces the shared-host prototype, this folder should be removed.
